# Gestion Ciment Backend

Backend complet pour la gestion d'une société de ciment (achats, ventes, stock, bonus fournisseurs, dépenses, etc.)

## Technologies
- Node.js + Express
- MySQL + mysql2
- JWT (authentification)
- bcrypt (hash mot de passe)
- Architecture MVC
